# UI Components Built With React And Chakra-UI

I love working with React and Chakra-UI and this is work in pogress project where I showcase UI components I have built for clients.

### Installing

Clone

```
git clone
```

Intall dependencies

```
yarn or npm i
```

start project

```
yarn dev or npm dev
```

## Running the tests

```
yarn test
```
